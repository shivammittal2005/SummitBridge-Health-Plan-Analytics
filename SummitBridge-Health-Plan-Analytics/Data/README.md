# Data

`SummitBridge-Case-Data.xlsx` contains the five source tables supplied for the case competition.

The data is included for reproducibility and portfolio review. It should be treated as fictional or competition-provided case data rather than real patient information.

## Source Tables

- `claims`
- `member_months`
- `enrollment`
- `providers`
- `avoidable_ed_reference`

Do not replace these files with real protected health information or personally identifiable information.
