# Dashboard

## Live Version

https://mc-kinsey-casecomp-web.vercel.app/

## Collaborative Source Repository

https://github.com/ananyamangal/Halo

The source code is maintained in the team's collaborative dashboard repository. The dashboard presents:

- Executive KPI cards
- High-cost-member prioritization
- Cost structure and monthly trends
- ED avoidability
- Specialty Rx concentration
- Network leakage and retention
- Scenario-based savings modeling
- Benchmarking and intervention monitoring
